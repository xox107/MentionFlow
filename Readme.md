MentionFlow 🚀
Social Intelligence & LinkedIn Mention Tracker

MentionFlow is a full-stack OSINT tool designed to track brand mentions and executive activity on LinkedIn. Specifically optimized for **Adya AI** and **Shayak Mazumder**.

✨ Features
- **Automated Scraper:** Python-based engine using Serper API.
- **Real-time Dashboard:** Clean, Dark-mode UI for monitoring mentions.
- **Intelligence Reports:** One-click PDF generation with wrapped links and clean formatting.
- **Time-Filtered:** Automatically tracks the last 180 days (6 months) of activity.

🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Engine:** Python 3.11, Serper API
- **Reporting:** FPDF2

🚀 Quick Start
1. Clone the repository.
2. Run `npm install` in the root.
3. Ensure your Python `venv` is active and run `pip install requests fpdf2`.
4. Double-click `run_app.bat` to launch.
